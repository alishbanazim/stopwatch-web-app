#  stopwatch-web-app

A modern and responsive stopwatch web application built using **HTML**, **CSS**, and **JavaScript**.

##  Live Demo

🔗 https://ticktrack-js.netlify.app

##  Features

- Start, Pause and Restart timer
- Real-time current date and time
- Dark / Light theme
- Theme preference saved using Local Storage
- Responsive design
- Animated restart button
- Clean and modern UI

##  Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
